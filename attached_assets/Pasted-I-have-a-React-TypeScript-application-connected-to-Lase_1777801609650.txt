2. BACKEND (Node.js + Express)
================================

Create AI endpoints:

--------------------------------
A) Smart Summary API
--------------------------------
POST /api/ai/summarize/:entryId

Steps:
1. Call Laserfiche API:
   GET http://localhost/LFRepositoryAPI/v1/Repositories/{repoId}/Entries/{entryId}?$select=*

2. Extract:
   - document name
   - metadata fields
   - dates
   - department
   - status

3. Clean metadata:
   If value = null → return ""

4. Send data to Ollama:

POST http://localhost:11434/api/generate

Body:
{
  "model": "llama3",
  "prompt": "
  حلل هذه الوثيقة:

  {metadata JSON}

  المطلوب:
  1. اكتب ملخص بسيط وواضح
  2. اشرح حالة الوثيقة
  3. هل توجد مشاكل او تناقضات؟
  4. اقترح تصنيف
  5. اقترح Tags
  ",
  "stream": false
}

5. Return AI response

--------------------------------
B) Natural Search API
--------------------------------
POST /api/ai/search

Input:
{
  "query": "user question"
}

Steps:

1. Fetch documents list from Laserfiche:
   GET /v1/Repositories/{repoId}/Entries/{folderId}

2. Extract for each document:
   - name
   - metadata
   - status
   - department
   - dates

3. Send to Ollama:

Prompt example:
"
لدي قائمة وثائق:

{documents JSON}

سؤال المستخدم:
{query}

المطلوب:
- ابحث عن الوثائق المناسبة
- ارجع أفضل النتائج
- اشرح لماذا تم اختيارها
"

4. Return filtered results
